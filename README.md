# Трекер прочитанных книг

Консольное приложение на Python для учёта прочитанных книг. Данные хранятся в файле `books.json`.

## Как запустить

```
python main.py
```

Требуется Python 3.6+. Сторонние библиотеки не нужны.

## Возможности

```
=== Трекер прочитанных книг ===
1. Добавить книгу
2. Показать все книги
3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход
```

- **Добавить книгу** — вводите название, автора, оценку (1–5) и дату прочтения. Дубликаты не добавляются.
- **Показать все книги** — нумерованный список с оценкой и датой.
- **Средняя оценка** — среднее арифметическое по всем книгам.
- **Статистика по авторам** — сколько книг каждого автора в списке.
- **Удалить книгу** — выбор по номеру из списка.

## Пример работы

```
=== Трекер прочитанных книг ===
1. Добавить книгу
...
Выбор: 1

--- Добавить книгу ---
Название: Мастер и Маргарита
Автор: Булгаков
Оценка (1-5): 5
Дата прочтения (например, 2024-06-10): 2024-05-01
Книга "Мастер и Маргарита" добавлена!

Выбор: 2

--- Все книги ---
1. «Мастер и Маргарита» — Булгаков | Оценка: 5/5 | Прочитана: 2024-05-01
```
